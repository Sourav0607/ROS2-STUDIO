Changelog for package ros2_studio
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

0.1.0 (2025-03-26)
------------------
* Initial release
* Real-time performance monitor for ROS2 topics and nodes (CPU, memory, frequency)
* Bag recorder with multi-topic selection and custom save path
* Bag player with adjustable rate (0.1x-10x) and loop controls
* Bag to CSV converter with full rosbag2_py message deserialization
* System dashboard showing CPU, memory, disk, network, ROS2 entities, and processes
* ROS2 CLI extension: ``ros2 studio``
* Contributors: Sourav Hawaldar
